Introduction to Web Development

Web development involves the designing, structuring and styling of webpages. Using HTML to structure, CSS to style and Javascript to add other functions which may be of use.

This unit particularly interests me as the imagination is the limit when it comes to styling and structuring your own webpage. From changing colours, creating whacky and unorthodox layouts, to using Javascript to implement interesting functions within a webpage.

Information Systems

Information systems are hugely important to corporations and businesses as they are used to collect, process and analyse data. They usually consist of networks of people and computers which interact with each other.

SQL (Structure Query Language), is a popular programming language used specifically to interact with databases, to input, retrieve and analyse data. In the modern day these are essential to any company/business with ambition.
